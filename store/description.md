Garoon のスケジュール画面を、日々の業務でちょっと便利に使うための非公式 Chrome 拡張機能です。

【主な機能】
■ 現在時刻ラインの表示
日表示・週表示のスケジュール画面に、現在時刻を示す赤いラインを重ねて表示します。「今ちょうど何時か」「次の予定まであと何分か」が一目で分かるようになり、会議の開始時刻を逃しにくくなります。

■ 予定登録ダイアログからの ToDo 追加
カレンダーをクリックして表示される「予定の簡易登録」ダイアログに、新しく「ToDo」タブを追加します。タブを切り替えるだけで、その日の ToDo を Garoon の ToDo 一覧へ直接登録できます。

■ オプションで個別に ON/OFF
拡張機能のオプション画面から、各機能を個別に有効・無効化できます。必要な機能だけを使えます。

【対応環境】
サイボウズのクラウド版 Garoon (*.cybozu.com) のみ対応しています。
オンプレミス版 Garoon には対応していません。

【ご注意】
本拡張機能はサイボウズ社の公式製品ではありません。Garoon の DOM 構造に依存しているため、Garoon 側の UI 変更により一時的に動作しなくなる可能性があります。
